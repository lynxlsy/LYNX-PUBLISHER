    Sobre o Lynx Publisher
    
              
    
                Uma aplicação desktop desenvolvida para simplificar a publicação de projetos no GitHub. 
                Com um fluxo guiado passo a passo, o Lynx Publisher torna o versionamento acessível, 
                reduzindo a fricção do Git e facilitando o onboarding de novos desenvolvedores.
              
    
            
    

            
            
    
              
    Funcionalidades
    
              
      
                
    • 
                  Seleção de Projeto
                  Escolha pastas ou arquivos ZIP para publicar
                
      • 
                
    • 
                  Verificação Git
                  Análise automática do status do repositório
                
      • 
                
    • 
                  Gerenciamento de Repositórios
                  Conecte ou crie repositórios diretamente no GitHub
                
      • 
                
    • 
                  Controle de Versão
                  Commit, push, pull e gerenciamento de branches
                
      • 
                
    • 
                  Logs Orientados
                  Mensagens claras com ações sugeridas
                
      • 
                
    • 
                  Tutorial Integrado
                  Aprenda Git e GitHub dentro do aplicativo
                
      • 
                
    • 
                  Configuração .gitignore
                  Defina facilmente arquivos a ignorar
